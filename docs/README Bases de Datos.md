# Análisis de Base de Datos

## Empresa

Belladonna Estética es un centro de estética que ofrece servicios
como manicura semipermanente, lifting de pestañas, depilación con cera
y diseño de cejas.

## Información que maneja el negocio

El centro necesita gestionar:

- Clientes
- Servicios ofrecidos
- Empleados que realizan los tratamientos
- Citas o reservas de los clientes

## Entidades identificadas

Las entidades principales son:

- Cliente
- Servicio
- Empleado
- Cita

## Relaciones entre entidades

Un cliente puede tener varias citas.

Un empleado puede realizar varios servicios.

Un servicio puede realizarlo varios empleados.

Cada cita está asociada con:

- un cliente
- un servicio
- un empleado
