# Proyecto: Belladonna Estética

Belladonna Estética es un centro de estética ficticio que ofrece
servicios como manicura semipermanente, lifting de pestañas,
depilación con cera y diseño de cejas.

## El proyecto consiste en desarrollar:

- Un portal web corporativo del centro (HTML + CSS)
- Una base de datos que modela el funcionamiento del negocio
- Una aplicación Java que gestiona clientes, servicios y citas
- Documentación técnica del sistema

## Tecnologías utilizadas:

HTML, CSS, Java, MySQL, JDBC y GitHub.
